¿Cómo iniciar sesión en GitHub?

Para comenzar a crear una pagina web de forma gratuita vamos a utilizar GitHub pages pero, primero necesitas iniciar sesión en tu cuenta. Si aún no tienes una cuenta, puedes crear una de forma gratuita. A continuación, te explicamos cómo hacerlo:

Iniciar sesión en GitHub

  1. Visita el sitio web de GitHub.
  2. Haz clic en el botón "Sign in" o (Iniciar sesión) en la esquina superior derecha.
  3. Ingresa tu dirección de correo electrónico o nombre de usuario y tu contraseña.
  4. Haz clic en el botón "Sign in" para acceder a tu cuenta. a continuacion te mostramos un video para que aprendas como aserlo visualmente

Consejos adicionales

  • Si olvidas tu contraseña, puedes restablecerla haciendo clic en el enlace "Forgot password?" o (¿Olvidaste tu contraseña?) en la página de inicio de Github también te recomiendo guardar el link de tu Github en un blog de notas , calentario o cualquier parte donde puedas guardar el link para cuando quieras abrirlo
