¿Cómo cambiar archivos en GitHub?

Nota importante: Usa el modo escritorio si estás en un celular para acceder a todas las opciones.

En este tutorial aprenderás cómo modificar archivos en GitHub sin usar la terminal.

1. Editar un archivo directamente en GitHub

  1. Ve a GitHub e inicia sesión.
  2. Abre el repositorio donde está el archivo.
  3. Haz clic en el archivo que deseas modificar.
  4. Presiona el botón Editar (ícono de lápiz ✏️).
  5. Realiza los cambios que necesites.
  6. Escribe un mensaje en Commit changes describiendo el cambio.
  7. Haz clic en Commit changes para guardar.

2. Subir un nuevo archivo para reemplazar el anterior

  1. En el repositorio, haz clic en “Add file”.
  2. Selecciona “Upload files”.
  3. Arrastra y suelta el archivo o elige uno desde tu computadora.
  4. Si el archivo ya existe, GitHub lo marcará como un cambio.
  5. Describe el cambio y haz clic en Commit changes.

3. Restaurar un archivo a una versión anterior

  1. Abre el archivo en GitHub.
  2. Haz clic en History para ver los cambios anteriores.
  3. Selecciona la versión que deseas restaurar.
  4. Copia el contenido y edita el archivo con los pasos anteriores.
