Lección 1: Variables y console.log

En JavaScript, puedes usar la palabra clave let para crear una variable. Intenta crear una variable llamada nombre que almacene "Pedro" y muestra su valor usando console.log().

let nombre = "Pedro";
console.log(nombre);

Lección 2: Funciones

function saludar(nombre) {
    console.log("Hola " + nombre);

Lección 3: Condicionales

Los condicionales te permiten ejecutar código basado en ciertas condiciones. Escribe un condicional que verifique si una variable llamada edad es mayor o igual a 18 y muestra "Mayor de edad" en la consola.

let edad = 20;
if (edad >= 18) {
    console.log("Mayor de edad");
}
